## 4. Technical SEO Checklist

### Google Search Console Setup

**Status:** Previously submitted (per project memory). Verify the following:

1. **Verify ownership:** Go to https://search.google.com/search-console → verify verbyai.com via DNS TXT record or HTML file upload
2. **Submit sitemap:** In Search Console → Sitemaps → submit `https://verbyai.com/sitemap.xml`
3. **Check indexing:** Go to URL Inspection → test each key page:
   - `verbyai.com/` (homepage)
   - `verbyai.com/features`
   - `verbyai.com/pricing`
   - `verbyai.com/download`
   - `verbyai.com/blog` (and each blog post URL)
4. **Request indexing:** For any page showing "URL is not on Google", click "Request Indexing"
5. **Check coverage report:** Look for errors (404s, redirect chains, noindex issues)
6. **Enable email notifications:** Settings → Email preferences → turn on for all alerts

### Bing Webmaster Tools Setup

1. Go to https://www.bing.com/webmasters
2. Sign in with Microsoft account
3. Import from Google Search Console (fastest method) OR add site manually
4. Submit sitemap: `https://verbyai.com/sitemap.xml`
5. Verify all pages are indexed
6. Check for crawl errors
