3. Description of the Service

Verby is an AI-powered voice dictation desktop application for macOS and Windows. The application runs locally on your computer and uses your microphone to capture speech, which is then transcribed using OpenAI Whisper and optionally processed by AI language models for cleanup, formatting, or enhancement. The resulting text is injected at your cursor position in any application.

To function properly, Verby requires the following system permissions:

  • Microphone access — to capture your voice for transcription
  • Accessibility access — to inject transcribed text at your cursor position across applications
  • Screen recording access (macOS) — to detect the active application and cursor context

These permissions are used solely for the core functionality of the application. We do not use these permissions to collect, store, or transmit any data beyond what is necessary for transcription and text injection.
